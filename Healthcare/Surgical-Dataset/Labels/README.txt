This dataset was used for both object detection (1st folder), as well as in occlusion reasoning (2nd folder) in which on top means is not occluded and bottom means that the object is occluded.

For object detection:
0- Scalpel nº4
1- Straight Dissection Clamp
2- Straight Mayo Scissor
3- Curved Mayo Scissor

For occlusion reasoning:
0- bottom
1- top
